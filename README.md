# xDeepLearning
**不借助深度学习框架**，实现机器学习的重要算法，搭建可复用的模型构件，从底层构造框架，以便：

- 检验对算法理解是否准确

- 自由替换模型中的底层算法，取得一手实验结果

- 灵活复现paper、验证新方法的结论

- 落地场景的算法POC
